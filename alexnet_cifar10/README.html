AlexNet Architecture

AlexNet introduced several key innovations that became standard in deep learning:

  • ReLU Activation: Replaced sigmoid/tanh with ReLU for faster training
  • Dropout: Introduced dropout (p=0.5) to prevent overfitting
  • Data Augmentation: Random cropping and horizontal flipping
  • GPU Training: First successful use of GPUs for deep learning
  • Local Response Normalization: Normalization across feature maps

Key Contributions

AlexNet's success demonstrated that deep convolutional networks could learn hierarchical representations directly from raw pixel data, achieving unprecedented accuracy on large-scale image classification tasks.
